## 🔴 根因分析

### 根因一：平局暴政 · 六爻模型系统性偏差

6场比赛5场判平局，实际0场平局。这比6月21日（3/4判平）更加严重。

**核心问题：** 六爻断课中，只要世爻与应爻旺度接近、或动爻生应同时生世、或缺乏明确的克制关系，模型就倾向于输出"平局/均势"。但在世界杯小组赛第三轮这种**生死战**环境下，球队要么全力争胜、要么彻底出局——平局概率天然偏低。

**第三轮特殊因素未纳入模型：**
- 出线生死战 → 球队不愿接受平局
- 同时开球 → 双方实时调整策略
- 小组形势明确 → 平局对双方可能都没意义

### 根因二：主客队分配错误

预测数据中3场比赛的主客场与赛程实际相反：
- 预测：墨西哥(主) vs 捷克(客) → 实际：捷克(主) vs 墨西哥(客)
- 预测：韩国(主) vs 南非(客) → 实际：南非(主) vs 韩国(客)
- 预测：巴西(主) vs 苏格兰(客) → 实际：苏格兰(主) vs 巴西(客)

**影响评估：** 即使纠正主客方向，5场判平局仍然全错。主客颠倒不是主要根因，但加剧了偏差。

### 根因三：比分硬编码问题持续

唯一命中方向的摩洛哥vs海地：预测1-0，实际4-2。
- 进球数预测偏差：3球 → 6球
- 比分模型完全没有反映摩洛哥的进攻火力（最终4球）和海地的反击能力（2球）

### 根因四：赛前信息未融入

赛前明确可知的信息未被模型利用：
- 墨西哥已2连胜提前出线 → 可能轮换？→ 实际：全主力3-0大胜
- 巴西实力碾压苏格兰 → 不可能0-0 → 实际：3-0
- 卡塔尔已基本出局、波黑争出线 → 波黑必胜决心强 → 实际：3-1
- 韩国想出线必须赢 → 不会满足平局 → 实际：0-1输球

### 根因五：高把握(🟢73%)预测完全翻车

波黑vs卡塔尔判0-0平局，信心73%（🟢高把握），实际波黑3-1大胜卡塔尔。

复盘该卦：晋卦，官鬼发动生应爻(客队卡塔尔)，世爻旺度-2.5远低于应爻1.5 → 模型认为卡塔尔更强，但实际波黑碾压。这说明**卦象解读可能偏了**：官鬼临玄武生应，可能是"裁判/暗算因素有利于客队"而非"客队实力强"。

## 🎯 改进建议

### 紧急修复
1. **六爻平局阈值大幅上调** — 当前几乎任何非明确克制的格局都判平局，需要至少世应旺度差>1.5且动爻明确指向一方才判胜负
2. **引入小组形势权重** — 第三轮"必须赢"的比赛，平局概率自动降权
3. **比分模型引入ELO/泊松** — 放弃硬编码比分，交给统计模型

### 架构反思
4. **六爻的足球预测瓶颈** — 连续两轮（6/21 + 6/25）的结论：六爻对足球比赛的胜负方向判断能力不足，平局偏差是系统性问题。建议：
   - 降低六爻权重，提升统计模型权重
   - 或完全弃用六爻方向判断，仅保留卦象作为"趣味彩蛋"
